![build status](https://img.shields.io/badge/build-passing-brightgreen)

## 介绍

vue ui 框架
## 开始使用

1. 安装
```
npm i raind-ui -S
```
2. 引入
```
import { Button } from 'raind-ui'
import 'raind-ui/dist/index.css'

export default {
  components: {
    'r-button': Button
  }
}
```
3. 引入svg symbols
```
<script src="//at.alicdn.com/t/font_2321457_7e55ja00emv.js"></script>
```
